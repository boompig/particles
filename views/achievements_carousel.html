Swagman

Swagman is a mesos framework meant for distributed testing of Yelp-main. It auto-scales as it is given more resources, and is meant to be both fault-tolerant and give better isolation for running tests. The documentation can be found here.

Buildbot manager status

Buildbot manager now has build status on the front page. This allows to see, at a glace, whether your build is done or not. Build status auto-refreshes, and is fetched lazily (only within the current viewport) for efficiency. Auto-update is disabled on window minimization or when the tab is in the background.

Buildbot run timeline

A visualization of a buildbot run, broken down by test case. Each row contains all the tests being run on a given builder. Each green box is a testcase. The testcases are ordered based on their actual interval of execution.

We used this view to get an idea of the bottlenecks in the current buildbot+testify setup, in the lead-up to the creation of Swagman
